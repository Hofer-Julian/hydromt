Models¶

HydroMT model plugins¶

HydroMT core can easily access and navigate through different model implementations thanks to its plugin architecture. The hydromt package contains core functionnalities and the command line interface. The models, following HydroMT’s architecture, are then implemented in separate plugins available in other python packages.
