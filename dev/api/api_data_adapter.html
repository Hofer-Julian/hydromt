Data Adapater

Data Catalog

General

data_adapter.DataCatalog([data_libs, logger])

Catalog of DataAdapter sources to easily read from different files and keep track of files which have been accessed.

data_adapter.DataCatalog.sources

Returns dictionary of DataAdapter sources.

data_adapter.DataCatalog.keys

Returns list of data source names.

data_adapter.DataCatalog.to_dict([...])

Return data catalog in dictionary format

data_adapter.DataCatalog.to_dataframe([...])

Return data catalog summary as DataFrame

data_adapter.DataCatalog.to_yml(path[, ...])

Write data catalog to yml format.

Add data sources

data_adapter.DataCatalog.from_artifacts([...])

Read a catalog file from https://github.com/DirkEilander/hydromt-artifacts releases.

data_adapter.DataCatalog.from_dict(data_dict)

Add data sources based on dictionary.

data_adapter.DataCatalog.from_yml(path[, ...])

Add data sources based on yml file.

data_adapter.DataCatalog.update(**kwargs)

Add data sources to library.

Get data

data_adapter.DataCatalog.get_rasterdataset(...)

Returns a clipped, sliced and unified RasterDataset from the data catalog.

data_adapter.DataCatalog.get_geodataset(...)

Returns a clipped, sliced and unified GeoDataset from the data catalog.

data_adapter.DataCatalog.get_geodataframe(...)

Returns a clipped and unified GeoDataFrame (vector) from the data catalog.
