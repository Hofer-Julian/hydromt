hydromt.stats.skills.percentual_bias

hydromt.stats.skills.percentual_bias(sim, obs, dim='time')[source]

Returns the percentual bias between two time series.

\[PBias= 100 * \frac{\sum_{i=1}^{N}(sim_{i}-obs_{i})}{\sum_{i=1}^{N}(obs_{i})}\]
Parameters

  • sim (xarray DataArray) – simulations time series

  • obs (xarray DataArray) – observations time series

  • dim (str, optional) – name of time dimension in sim and obs (the default is ‘time’)

Returns

percentual bias

Return type

xarray DataArray
