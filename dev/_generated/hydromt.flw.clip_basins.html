hydromt.flw.clip_basins¶

hydromt.flw.clip_basins(ds, flwdir, xy, flwdir_name='flwdir', **stream_kwargs)[source]¶

Clip a dataset to a subbasin.

Parameters

  • ds (xarray.Dataset) – dataset containing flow direction data

  • flwdir (pyflwdir.FlwdirRaster) – Flow direction raster object

  • xy (tuple of array_like of float) – x, y coordinates of (sub)basin outlet locations

  • flwdir_name (str, optional) – name of flow direction DataArray, by default ‘dir’

  • stream_kwargs (key-word arguments) – name of variable in ds and threshold value

Returns

clipped dataset

Return type

xarray.Dataset
